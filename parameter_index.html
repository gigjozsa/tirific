Unsupported parameters for GIPSY

We provide old TiRiFiC versions that still depend on GIPSY. This concerns TiRiFiC until version 2.1.5. Here we list keywords that are only supported for version 2.2.0 and higher.

Name Category Unit Description
COOLBIN I/O (default 1)   Number of voxels along each axis per input voxel for spatial 3D model as specified with COOLGAL (default: 1)
DEFFILE I/O (not required)   Name of default file
INDINTY Fitting (default 1)   Interpolation scheme between parameters indexed by VARINDX
GR_INTERPAD_i_j= I/O (defaults to INTY)   Interpolation scheme for lines jth additional series ith plot; 0: linear, 1: natural cubic spline, 2: natural Akima
GR_INTER_i= I/O (defaults to INTY)   Interpolation scheme for lines for main data series; 0: linear, 1: natural cubic spline, 2: natural Akima
INTY Geometrical (default 0)   Interpolation scheme for sub-grid (sub-ring) parameters; 0: linear, 1: natural cubic spline, 2: natural Akima
PROMPT I/O (not required)   Prompt on error (1: yes, 0: no)?
RESTARTNAME Fitting (optional)   Give file name for restarting TiRiFiC
RESTARTID Fitting (optional)   Give ID for restarting tirific
