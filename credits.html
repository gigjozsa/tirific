TiRiFiC has originally been designed and implemented by Gyula I. G. Józsa, Franz Kenn, Thomas A. Oosterloo, and Ulrich Klein. Currently, TiRiFiC is mainly being maintained and developed by Gyula I. G. Józsa, with contributions and suggestions from many people. Apart from the original authors, we would like to mention:

  • Hyun-Jin Bae (prototype parallel implementation of TiRiFiC)
  • Peter Kamphuis (testing TiRiFiC, developing TiRiFiC pipeline, programming TiRiFiC output)
  • Se-Heon Oh (testing TiRiFiC, developing TiRiFiC pipeline)
  • George Heald (Python default file editor)
  • Phillip Cigan (prototype description of earlier TiRiFiC versions)
  • Erwin de Blok (advise documentation)
  • Sungsoo Kim, Aeree Chung, the HALOGAS team, the WALLABY galaxy parametrisation group (testing, advise)
  • Stephan Peters (advise fitting techniques)
  • Milan den Heijer implementation fitting and alternative parametrisation

The appropriate TiRiFiC reference in a publication is Józsa, G. I. G., Kenn, F., Klein, U., Oosterloo, T. A. 2007, A&A, 468, 731. While TiRiFiC has evolved quite a bit since then, the basic technique is described appropriately in this paper.

If you want to mention TiRiFiC in a talk, you may want to show the TiRiFiC logo and a link to this web page (http://gigjozsa.github.io/tirific/).
